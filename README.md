# Anna Montero — Portfolio 2026

Design engineer portfolio. North star: **build proof, not just knowledge.**

## Stack

- Next.js (App Router)
- Tailwind CSS v4
- TypeScript
- pnpm

## Develop

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3026](http://localhost:3026).

## Ship

```bash
pnpm build
pnpm start
```

**Live:** [portfolio-2026-six-gilt.vercel.app](https://portfolio-2026-six-gilt.vercel.app/)
